使用Ts的原因

js是一门弱类型、动态类型的语言，存在类型隐式转换、运行时报错等难以预知的问题；而ts是静态类型、强类型语言，编译时即可报错，类型检查更为严格；使开发更为严谨，错误更易发现；其次，接口的定义以及类型的定义更利于团队中代码的维护，也便于代码的阅读
